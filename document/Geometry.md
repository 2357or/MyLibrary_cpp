# Geometry Library
面幾何学平計算ライブラリ

<b>*Vector2 に依存します*</b>

関数の一括挿入: `:Library geom/all`

サンプル: [Geometry.cpp](/sample/Geometry.cpp)

## 関数
| 名前      | コマンド             | 関数名                           |
| :-----    | :---------           | :--------                        |
| 2点間距離 | `:Library geom/dist` | Distance(Vector2, Vector2) -> ld |
| 内積      | `:Library geom/dot`  | Dot(Vector2, Vector2) -> ld      |
